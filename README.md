# 智能作业批改系统 (IHGS WeChat Mini Program)

本项目是一个基于微信小程序的智能作业批改系统，旨在连接教师与学生，提供便捷的作业布置、提交、批改及学习统计功能。

## 📋 功能概览 (Feature Overview)

### ✅ 已完成的功能 (Completed Features)

#### 1. 用户认证与管理 (Authentication & User Management)
*   **微信登录**: 支持完整的微信授权登录流程，自动获取用户头像与昵称，并基于 JWT (Access/Refresh Token) 进行会话管理。
*   **个人信息管理**:
    *   用户可查看并编辑个人资料（头像、昵称、性别、角色）。
    *   **角色系统**: 支持区分“学生”与“教师”两种角色，界面根据角色动态展示不同功能入口。
*   **学习统计 (学生端)**: 提供可视化的学习数据统计，展示总做题数及知识点掌握情况。

#### 2. 班级系统 (Class System)
*   **班级创建 (教师端)**: 教师可以创建新班级，生成唯一的班级码。
*   **班级加入 (学生端)**: 学生可通过输入班级码加入指定班级。
*   **班级列表**: 首页及班级管理页面清晰展示用户关联的所有班级。
*   **班级详情**: 展示班级基本信息（教师、创建时间、描述）及最近发布的作业概览。

#### 3. 作业系统 (Homework System)
*   **作业列表**:
    *   支持按状态筛选作业：全部、待完成、已提交、已批改。
    *   清晰展示作业截止日期、提交状态及分数。
*   **做题功能 (学生端)**:
    *   支持多种题型（选择题、主观题）。
    *   支持图片上传功能，方便提交纸质作业照片。
*   **作业管理 (教师端)**: 教师可查看班级下的作业列表及学生的提交情况。

#### 4. 题库与练习 (Question Bank & Practice)
*   **题目搜索**: 提供强大的搜索功能，支持按关键字或知识点筛选题目。
*   **自主练习**: 学生可随时进行随机练习或针对特定题目进行强化训练。

### 🚧 正在进行中的功能 (Features Under Development)

#### 1. 班级管理增强 (Advanced Class Management)
*   **学生管理**: 教师端直接管理班级成员（移除学生、查看学生详情）的功能目前标记为 **"不可用"**。
*   **班级互动**:
    *   **班级通知**: 发布和查看班级公告的功能尚在开发中。
    *   **课程表**: 班级专属课程表功能尚未上线。
    *   **详细数据**: 教师查看班级详细统计数据的入口目前标记为 **"不可用"**。

#### 2. 智能作业布置 (Smart Assignment)
*   **AI 拍照识别**: 计划引入 AI 技术，支持教师拍照自动识别题目并生成电子版作业，目前状态为 **"未开放"**。
*   **手动录入优化**: 手动输入题目（支持复杂公式和多媒体）的功能目前处于 **"测试中"** 阶段。
*   **作业管理升级**: 教师端的班级作业深度管理功能（如批量批改、详细数据分析）目前标记为 **"测试中"**。

#### 3. 提交与反馈 (Submission & Feedback)
*   **提交记录查看**: 学生查看自己历史提交的具体内容及详细批改反馈的功能目前标记为 **"暂不可用"**。
*   **系统设置**: 首页的系统全局设置入口尚未启用。

## 🛠️ 技术栈 (Tech Stack)
*   **前端**: 微信小程序原生框架 (WXML, WXSS, JS, JSON)
*   **组件库**: WeUI / 自定义组件
*   **网络**: `wx.request` 封装，支持拦截器与 Token 刷新机制
