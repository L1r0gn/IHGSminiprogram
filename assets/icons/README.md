# TabBar 图标说明

此目录用于存放底部导航栏的图标文件。

**需要的图标文件（建议尺寸：81px x 81px）：**

1. `home.png` - 首页图标（灰色）
2. `home-active.png` - 首页图标（高亮色）
3. `courses.png` - 课程图标（灰色）
4. `courses-active.png` - 课程图标（高亮色）
5. `analysis.png` - 分析图标（灰色）
6. `analysis-active.png` - 分析图标（高亮色）
7. `profile.png` - 我的图标（灰色）
8. `profile-active.png` - 我的图标（高亮色）

**临时解决方案：**
- 当前配置中引用了这些图标文件
- 如果没有实际图标文件，小程序将无法正常启动
- 建议使用以下在线工具生成图标：
  - https://iconfont.cn/
  - https://www.flaticon.com/

**图标设计建议：**
- 首页：🏠 房子图标
- 课程：📚 书本图标
- 分析：📊 图表图标
- 我的：👤 用户图标
